# Turf Ground Booking System
##description 
This is a C-based Turf Ground Booking System developed using:
- Structures
- Pointers
- File Handling
- Dynamic Memory Allocation

## Features
- Admin Login
- View Available Turfs
- Book a Turf
- View Booking Details
- Generate Bill
- Store Booking Data in File

## Technologies Used
- C Programming
- File Handling
- Structures
- Pointers
